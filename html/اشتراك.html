صفحة اشتراكات التغذية الشاملة

باقة الصحة اليومية - Basic Daily

  • ✔ خطة غذائية أسبوعية مصممة خصيصًا
  • ✔ اختيار وجبتين صحيتين يوميًا (غداء + عشاء)
  • ✔ خيارات متنوعة (نباتي / قليل الكربوهيدرات / خالي من الغلوتين)
  • ✔ توصيل يومي داخل المدينة
  • ✔ تواصل أسبوعي مع أخصائي التغذية

💰 السعر: 50 $

📦 الوجبات: 14 وجبة أسبوعيًا (غداء + عشاء)

🕓 مدة الاشتراك: شهر / قابل للتجديد

باقة الرياضي المتوازن - Active Fit

  • ✔ خطة غذائية شهرية حسب هدفك (تنشيف، تضخيم، ثبات)
  • ✔ 3 وجبات صحية يوميًا (إفطار + غداء + عشاء)
  • ✔ توصيل يومي + تحديث أسبوعي للوجبات حسب الأداء
  • ✔ جلسة شهرية مع أخصائي تغذية وتمارين
  • ✔ إمكانية تخصيص الوجبات حسب المتطلبات الغذائية

💰 السعر: 70 $

📦 الوجبات: 21 وجبة أسبوعيًا

🕓 مدة الاشتراك: شهر أو 3 أشهر مع خصم

باقة العناية الشاملة - Wellness Plus

  • ✔ خطة تغذية شاملة + برنامج دعم نفسي غذائي
  • ✔ 3 وجبات + سناك صحي يوميًا
  • ✔ جلسات Zoom أسبوعية مع الأخصائيين
  • ✔ دعم مستمر على مدار الساعة عبر WhatsApp
  • ✔ تقارير متابعة وتحليل شهري (InBody أو بديل)
  • ✔ هدايا غذائية + تخفيضات على منتجات صحية

💰 السعر: 100 $

📦 الوجبات: 28 وجبة + 7 سناك أسبوعيًا

🕓 مدة الاشتراك: من شهر حتى 6 أشهر

باقات التدريب الرياضي

باقة المبتدئ - Start Fit

  • ✔ جدول تمارين منزلي لمدة شهر (3 أيام بالأسبوع)
  • ✔ فيديوهات توضيحية + تعليمات لكل تمرين
  • ✔ دعم عبر البريد أو WhatsApp أسبوعيًا
  • ✔ خطة تمارين بسيطة بدون معدات

💰 السعر: 50 $

🕓 المدة: شهر واحد

باقة الرياضي النشيط - Active Pro

  • ✔ خطة تمارين متكاملة (منزلي أو جيم)
  • ✔ 5 أيام تمرين أسبوعيًا
  • ✔ متابعة عبر Zoom مرتين بالشهر
  • ✔ تواصل مباشر مع المدرب لتعديل الخطة حسب الأداء
  • ✔ دمج مع خطة غذائية عند الطلب

💰 السعر: 70 $

🕓 المدة: شهر أو 3 أشهر

باقة التدريب الشخصي - Premium Coach

  • ✔ تدريب شخصي أونلاين مع مدرب/مدربة (جلسات أسبوعية)
  • ✔ جدول تمارين مخصص حسب الهدف والوقت المتاح
  • ✔ تقارير تقدم أسبوعية + فيديوهات متابعة
  • ✔ دعم دائم عبر WhatsApp
  • ✔ تقييم وضع الجسم قبل وبعد + متابعة غذائية اختيارية

💰 السعر: 150 $

🕓 المدة: اشتراك شهري أو باقات 3 و6 أشهر
